CI/CD

  • docker pull thoba - копируем образ приложения.
  • docker images - проверим, появился ли скопированный образ в списке образов.
  • docker run -dp (порт,запуска приложения):(порт, который требует инструкция)/(id_контейнера) - запустить контейнер с образом.
  • docker ps - вывод списока всех запущенных контейнеров .
  • docker stop id_контейнера - остановка контейнера .
